Questa sezione approfondisce la configurazione WordPress/SiteGround, gli accessi e i componenti tecnici da monitorare.

## Architettura WordPress

- Installazione gestita con SiteGround Site Tools (WordPress Toolkit).  
- Tema personalizzato basato su Elementor + template brand BF Wellness.  
- Database MySQL (credenziali salvate in `wp-config.php` e custodite dal responsabile IT).

## Accesso e credenziali

<Card title="Gestione accessi" icon="key">
  - SiteGround Site Tools: account amministrativi gestiti dal Responsabile IT.  
  - WordPress admin: creare utenti nominativi (no credenziali condivise).  
  - FTP/SFTP: credenziali generate on-demand, condivise tramite canali sicuri.  
  - Accesso DB (phpMyAdmin): fornito temporaneamente dal Responsabile IT per interventi mirati.
</Card>

## Sicurezza

- Abilitare 2FA sugli account WordPress amministrativi (plugin consigliato).  
- Limitare tentativi di login (plugin dedicato o funzionalit√† SG).  
- Monitorare log accessi e file changes tramite SiteGround Security.  
- Pianificare vulnerability scan trimestrale.
