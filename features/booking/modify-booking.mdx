# Modificare una Prenotazione

Modificare una prenotazione esistente.

## Capabilities

- Change date/time
- Change worker(s)
- Update service
- Add notes

## User Flow

1. Navigate to booking details
2. Tap "Modify"
3. Select what to change
4. Confirm changes
